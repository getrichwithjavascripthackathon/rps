
## Project setup

```bash
npm install
```

Set the credential environemtn variables:

```bash
export GMAIL__USER="gmail-user-to-send-mail-as@gmail.com"
export GMAIL__PASSWORD="gmail-user's-password"
```

## Running the project

```bash
node server.js
open http://localhost:9400
```
