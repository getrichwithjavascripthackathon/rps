
## Project setup

```bash
npm install
```

Create the file `credentials.json` in the project folder that looks like this:

```json
{
  "gmail": {
    "user": "gmail-user-to-send-mail-as@gmail.com",
    "password": "gmail-user's-password"
  }
}
```

## Running the project

```bash
node server.js
open http://localhost:9400
```
